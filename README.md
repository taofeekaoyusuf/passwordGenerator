# PASSWORD GENERATOR APPLICATION

### This is a Password generator application that helps generate passwords in different flavours and lengths.

<img width="1585" height="915" alt="image" src="https://github.com/user-attachments/assets/590d4f14-bb43-4d8b-8f5b-31af82a469b2" />

Password generation selection could be in different forms, as follows:
  - Alphabets | Numerics | Special Characters only.
  - Alpha-Numeric
  - Alpha-Special Character, or
  - Numeric-Special Character

And it could also be of either 5, 10, or 15 in length/size.
